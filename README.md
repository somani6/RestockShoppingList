# RestockShoppingList

RestockShoppingList ist ein World of Warcraft Addon, das hilft, Einkaufslisten für **Auctionator** automatisch zu verwalten und aufzufüllen.

## Features
- **Listenverwaltung:** Erstellen, Umbenennen und Löschen von Einkaufslisten.
- **Item-Konfiguration:** Hinzufügen von Items mit Wunschmenge und Qualität.
- **Auctionator Integration:** Exportiert die Listen direkt in das Einkaufslisten-Format von Auctionator, wenn das Auktionshaus geöffnet wird.
- **Import/Export:** Teile deine Listen mit anderen Spielern über generierte Strings (Base64).

## Installation
1. Lade das Addon herunter.
2. Entpacke den Ordner nach `Interface\AddOns\RestockShoppingList`.
3. Stelle sicher, dass Auctionator installiert ist.

## Befehle
- `/rsl` - Öffnet oder schließt das Konfigurationsfenster.
